Bourbachiste Language
An implementation of the formal Bourbaki language developed in their "Elements of Mathematic" 

Plan:
1. Parsing - how does one even parse an input language?
2. How does the Bourbaki language work and how are proofs created
3. The special aspects of Bourbaki - structures

NB: This is a trial project based on (arguably) outdated work - I believe that Type Theories and Category Theories have superseded Bourbaki, and have significantly more collaborators. For examples, check out Agda and Lean.
